# Network-Sniffer-IDS
##Abstract
The Intelligent Real-Time Network Intrusion Detection and Traffic Analysis System is a cybersecurity monitoring platform designed to detect malicious activities in network traffic using a hybrid detection approach. The system combines packet sniffing, deep packet inspection (DPI), rule-based intrusion detection, and machine learning techniques to provide accurate and real-time threat identification. The primary objective of the project is to develop a prototype capable of monitoring live network traffic, identifying suspicious behavior, and presenting the results through an interactive visualization dashboard.
The system captures live packets from the network and extracts essential metadata such as source and destination addresses, protocols, ports, and packet sizes. Deep Packet Inspection is used to analyze protocol level information including HTTP requests, DNS queries, and TCP flags. A rule-based detection module identifies known attacks such as port scanning, SYN flooding, and ICMP flooding based on predefined traffic patterns.
To enhance detection capability, the system integrates machine learning models trained on the CIC-IDS-2017 dataset. A Random Forest classifier is used to identify known attack types, while an Isolation Forest model detects anomalous or previously unseen behaviors. A feature extraction layer converts packet streams into statistical flow-based features suitable for machine learning analysis.
The platform includes a real-time monitoring dashboard developed using PySide6, which visualizes network statistics, bandwidth usage, and security alerts. By combining traditional intrusion detection methods with artificial intelligence, the system provides an effective approach for identifying both known and zero-day network attacks. This project demonstrates the feasibility of building an intelligent network monitoring tool capable of supporting modern cybersecurity operations.
