# Abstract

The rapid evolution of cyber threats necessitates advanced, real-time mechanisms for network monitoring and intrusion detection. This project presents an Intelligent Real-Time Network Intrusion Detection and Traffic Analysis System, designed as a comprehensive hybrid platform for modern cybersecurity monitoring. The system integrates traditional signature-based detection with advanced machine learning capabilities to identify both known and zero-day attacks. 

Live network traffic is continuously captured and subjected to deep packet inspection (DPI) and protocol analysis. The extracted flow data is processed through a robust feature extraction layer that computes essential time-series metrics. For threat identification, the system utilizes a dual-engine architecture: a rule-based Intrusion Detection System (IDS) for immediate recognition of established malicious patterns (e.g., port scans, flood attacks), and a machine learning engine evaluating traffic anomalies. The ML engine leverages a Random Forest classifier trained on the CIC-IDS-2017 dataset to detect specific attack signatures, alongside an Isolation Forest model to flag statistical deviations indicative of zero-day vulnerabilities. 

Culminating in a dynamic, PySide6-based graphical user interface, the platform provides security personnel with operational visibility through live analytics, bandwidth monitoring, and a normalized threat alert feed. By combining deterministic rules with adaptive artificial intelligence, this prototype delivers a resilient, modular, and high-performance solution for real-time network defense and traffic analysis.
